# quantumqueen-ios

iOS WKWebView wrapper for **The Queen of Quantum Chaos** PWA (https://heuremen.org/quantum-queen/), generated by [PWABuilder](https://www.pwabuilder.com/) and built via [Codemagic](https://codemagic.io/).

## App metadata

- **App name:** The Queen of Quantum Chaos
- - **Short name:** Quantum Queen
  - - **Bundle ID:** `org.heuremen.quantumqueen`
    - - **Hosted URL:** https://heuremen.org/quantum-queen/
      - - **Category:** Education / Games
        - - **Description:** Six stories. Six laws. One coin at the bottom. A choose-your-own-adventure through quantum physics.
          - - **Keywords:** quantum, physics, education, CYOA, choose your own adventure, science, kids

Web users still see them; installed-app users do not. So the iOS app contains no outbound third-party links.
